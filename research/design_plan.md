# Adaptive Math Tutor Design Plan

## Goal

Build an AI mathematics tutor that combines:

- grading
- reasoning diagnosis
- adaptive hints
- minimal intervention

## Design Principle

The system should avoid being only an answer generator or strict grader.

Instead:

Student Work
↓
Diagnosis
↓
Feedback Planning
↓
Adaptive Response


## Core Components

### 1. Solution Diagnosis

The system identifies:

- correct/incorrect reasoning
- first error step
- error type
- valid alternative methods


### 2. Feedback Generation

Feedback should adapt based on student needs:

- confirmation
- conceptual hint
- next operation
- worked example
- complete solution when requested


### 3. Learning Evaluation

Success is measured by whether students improve after feedback.
